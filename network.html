Chapter 4: Networking & the Internet

TERM DEFINITION
Network Classifications
  • LAN- local area network
  • MAN- metropolitan area network
  • WAN- wide area network
Network Models
  • bus network- refers to the pattern in which machines are connected
  • star network- popular with wireless networks
Protocols
  • CSMA/CD Ethernet- carrier sense, multiple access with collision detection
  • CSMA/CA wifi- carrier sense, multiple access with collision avoidance
Combining Networks
  • repeater- a device that passes signals between two original buses
  • bridge- connects two buses, but does not pass all messages across the connection
  • switch- essentially a bridge with multiple connections (several buses)
  • router- handles connections between networks to form an internet
  • gateway- the point at which one network is linked to an internet
Client/server model
  • P2P: peer-to-peer- involves processes that provide service to and receive service from each other
  • distributed system- consists of software units that execute as processes on different computers
  • cloud computing- where huge pools of shared computers on the network can be allocated as needed
Internet Addressing
  • Tier 1- very high speed, high capacity, international WANs
  • Tier 2- more regional in scope and less potent in their capabilities than tier 1
  • Tier 3- access provided to the internet network when tier 1 fails
IP Address
  • ICANN
  • DNS- domain name system
  • TLDs- top-level domain
  • Country-code TLDs- code specific to each country's internet
Internet Protocols
  • FTP- file transfer protocol used for accessing another computer from a great distance
  • HTTP- more sophisticated web servers and browsers
  • SMTP- simple mail transfer protocol
World Wide Web
  • URL- Uniform Resource Locator
  • client-side activities- activities that are performed by a client or a browser
  • server-side activities- activitites performed by a web server
Internet four-layer software
  • application layer- where a message typically originates
  • transport layer- where a message is prepared for transmission
  • link layer- where a message is transmitted and received
Internet is:
  • packet-switching network- where data is broken down into blocks of bits called packets
  • fault tolerant- where data can have missing information and still run
  • redundant- where data can be repeated
Cybersecurity
  • malware- attacks that incorporate the use of malicious software
  • virus- software that infects a computer by inserting itself into computer programs
  • worm- an autonomous program that transfers itself through a network and copies itself into other computers
  • Trojan horse- a program that enters a computer system disguised as a desirable program
  • spyware- software that collects information about activities of the computer and reports back to the instigator
  • phishing- a technique of obtaining information by simply asking for it
  • DDoS attack- (distributed denial of service) when the work of generating the attacks is distributed over many hosts
Protection
  • firewall- a technique used to filter traffic passing through a point in the network
  • spam filter- firewalls designed to block unwanted emails
  • proxy server- a software unit that acts with the goal of shielding the client
  • antivirus software- used to detect and remove viruses and other infections
  • cryptography- the study of sending and receiving messages securely
  • public key encryption- involves techniques using secure encoded messages
