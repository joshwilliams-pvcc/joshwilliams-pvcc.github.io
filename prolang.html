Chapter 6:Programming Languages

TERM DEFINITION
machine language an encoding system that consists of sequences of instructions
debugging locating and correcting errors
assembler programs developed to convert mnemonic expressions into machine language instructions
assembly language a mnemonic system for representing programs
translator a program that translates programs into machine-language programs
compiler programs that compile several machine instructions into short sequences
interpreter an alternative to translators that implement third-generation languages
cross-platform software programs that operate independently as well as machine independent
natural language languages such as English, German, and Latin
formal language languages such as programming languages
machine independence the development of third-generation languages so they can be used on any machine
high-level programming language provide more abstractions for the programmer and make it easier to read and write a program
object-oriented programming (OOP) software capable of performing actions that are related to itself/other objects
scripting language a subset of the imperative programming languages
data types:
  • integer- numeric data consisting of whole numbers
  • float- data that might contain values other than whole numbers
  • string- a strand of data
